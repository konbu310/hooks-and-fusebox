### Reference

### Dependencies

* react@16.8
* react-dom@16.8
* fuse-box@4.0.0
* typescript@3.3.1

### Build

```bash.command.sh
git clone https://github.com/konbu310/hooks-and-fusebox.git
cd hooks-and-fusebox
yarn
make dev # production build -> make build
```
Now, you can see a result at http://localhost:4444
